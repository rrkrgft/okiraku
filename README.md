# 卒業課題
## 開発言語バージョン
 - rails 6.1.7
 - ruby 3.0.1

## 就業Termで学んだ技術
 - device
 - お気に入り機能

## カリキュラム外技術
 - cancancan
 - ransack

 ## アプリケーションの実行手順
  `git clone git@github.com:rrkrgft/okiraku.git`  
  `cd okiraku`  
  `rails db:create`  
  `bundle install`  
  `yarn install`  
  `rails s`
